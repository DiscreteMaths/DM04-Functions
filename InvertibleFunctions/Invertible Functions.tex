\frametitle{Invertible Functions}
\Large
\vspace{-2cm}

\begin{itemize}
\item A function is \textbf{onto} if is range is equal to its codomain.

\item A function is \textbf{one-to-one} if no two distinct elements of the domain have the same image.
\end{itemize}
